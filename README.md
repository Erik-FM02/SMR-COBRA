# SMR-COBRA 🐍

Repositorio colaborativo creado para una actividad del ciclo formativo **Sistemas Microinformáticos y Redes (SMR)**.

El objetivo de este proyecto es aprender a trabajar con **Git y GitHub**, utilizando repositorios compartidos para organizar información y colaborar entre el equipo.

---

## 📌 Objetivos del proyecto

* Aprender a utilizar **GitHub** como plataforma de trabajo colaborativo.
* Practicar la creación de archivos y carpetas dentro de un repositorio.
* Comprender el funcionamiento básico del **control de versiones**.
* Desarrollar habilidades de trabajo en equipo en proyectos informáticos.

---

## 📂 Estructura del repositorio

El repositorio está organizado de la siguiente forma:

```
SMR-COBRA
│
├── README.md
├── alumno.txt
│
├── NombreAlumno1
│   └── informacion.txt
│
├── NombreAlumno2
│   └── informacion.txt
│
└── NombreAlumno3
    └── informacion.txt
```

### Contenido

**README.md**
Archivo principal del repositorio. Explica el propósito del proyecto y cómo está organizado.

**alumno.txt**
Archivo compartido donde cada uno añade su nombre en una línea independiente.

**Carpetas de alumnos**
Cada uno de nosotros debe crear una carpeta con su nombre y añadir dentro un archivo explicando:

* Por qué le gusta la informática.
* Qué aspecto de la informática le parece más interesante.

---

## 👨‍💻 Participantes

Los alumnos participantes deben añadir su nombre en el archivo **alumno.txt**.

---

## 🛠 Tecnologías utilizadas

Este proyecto utiliza herramientas básicas del entorno de desarrollo moderno:

* **Git** – Sistema de control de versiones.
* **GitHub** – Plataforma para alojar y colaborar en proyectos.
